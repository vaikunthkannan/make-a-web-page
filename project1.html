Lesson 1: Introduction to HTML

How do we get to see webpages using our computer

When we input the address of a webpage in the computer it send a HTTP request to the server and the server
finds the appropriate document and returns it back to the computer which in turn is displayed by the browser.

Tags

HTML files can contain a variety of tags including b - bold; a - anchor; em - emphasis; br - break and so on. These tags helps in identifying the type of element that is being used. Further explanation can be found in this video. Click this link for HTML Tags.

Inline and Block Elements

Some tags used in HTML create a block around the content for which the various dimensions could be specified. Few examples are the paragraph tag and div tag. The break tag and other tags used are mostly inline since they do not create any invisible block around the content.

Lesson 2:Creating a Structured Document with HTML

Document Structure

Usually an HTML document must contain the opening html tag followed by the head tag under which the title and other meta data are entered and displayed in the top of the web page. The body contains the content of the web page and the heading tags can be used to specify the size of the heading. Indentations are used to identify the opening and closing of tags and also makes it easier for humans to read and intrepret the code.

Boxes and Pages

In a browser, if we inspect each element of the page we are looking at, we see that all the element are made up of boxes, sometimes boxes within boxes. When we read an html document, each indented element corresponds to the stage in which it is inside the box.

Text Editing Programs

There are different programs used to type and edit html code. Some editors like Sublime(which I use) makes our life easier by generating closing tags for every tag we open. Other editors can also be used based on the convenience of the user.

Dissect a Page into boxes

When we try to design a page using HTML and CSS, its easier to imagine each and every element as a box and then write the code based on the number of boxes we require and the content inside each box. CSS elements allow us to add styles to the page like background colour, font colour, box width, size, etc.

Adding style to HTML using CSS

Avoid repetition

Trying to avoid repetition of code helps us in identifying errors in the code and fixing them easily. If same lines of code is repeated again and again, an error needs to be rectified at all the places and this is time consuming.

Get to know about CSS

CSS stands for Cascading Style Sheets. An HTML document contain various CSS sheets linked to it. We can use the link tag to link the CSS document with the html one. All attributes of the html page can be given using CSS and it is usually easier done by specifying each class used in the html document.

Positioning and Flex

If we specify the width of every div in the html page it results in boxes which are stacked one below another. We need to position the boxes and one useful concept covering this is the use of flex box layout. Flex box provides an efficient way to layout and display elements next to each other.
